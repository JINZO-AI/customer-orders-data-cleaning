# Customer Orders Data Cleaning

A data cleaning project using Python and Pandas on a real-world messy customer orders dataset.

## Project Overview

Raw customer order data is rarely clean. This project tackles common real-world data quality issues including missing values, duplicates, inconsistent formatting, and incorrect data types — transforming a messy dataset into a clean, analysis-ready version.

## Dataset

- **File:** `messy_customer_orders.csv`
- **Size:** ~50,000 rows
- **Source:** Real-world style customer orders data

## Tools & Libraries

- Python 3
- Pandas
- NumPy
- ydata-profiling (automated EDA report)
- Google Colab

## What Was Done

- Generated automated profiling report to understand data quality issues
- Removed duplicate records
- Handled missing / null values
- Standardized column formats (dates, strings, numeric types)
- Exported clean dataset for analysis

## Files

1. Open `data_cleaning.ipynb` in Google Colab or Jupyter Notebook
2. Upload `messy_customer_orders.csv` when prompted
3. Run all cells in order
